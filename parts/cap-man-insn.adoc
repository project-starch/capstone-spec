== Revocation

[#revcap-creation]
=== Revocation Capability Creation

The `MREV` instruction creates a revocation capability.

[wavedrom,,svg]
....
{reg: [
    {bits: 7, name: '0b1011011'},
    {bits: 5, name: 'rd' },
    {bits: 3, name: '0b001' },
    {bits: 5, name: 'rs1' },
    {bits: 5, name: '*' },
    {bits: 7, name: '0b0001000' }
]}
....

[#rev-operation]
=== Revocation Operation

The `REVOKE` instruction revokes a capability.
