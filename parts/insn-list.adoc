.Note
****
*For instruction operands:*

I:: Integer register
C:: Capability register
-:: Not used

*For immediates:*

S:: Sign-extended
Z:: Zero-extended
-:: Not used

*For worlds:*

N:: Normal world
S:: Secure world
*:: Either world

*For variants:*

P:: _Pure Capstone_
T:: _TransCapstone_
*:: Either variant
****
